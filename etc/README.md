Windows版鳴潮で顔が動かない人用。

編集するファイルがある場所。デフォルトだとここにあるはず。

C:\Wuthering Waves\Wuthering Waves Game\Client\Saved\Config\WindowsNoEditor

ここにあるEngine.iniとDeviceProfiles.iniを、おのおのこのページに書いてあるもので上書きして保存。

https://github.com/deki-haibane/underwater/blob/main/etc/DeviceProfiles.ini

https://github.com/deki-haibane/underwater/blob/main/etc/Engine.ini

万が一おかしくなった場合はファイル自体を削除してしまって大丈夫です、次回起動時にデフォルトのものが自動で作られます。

つよつよPCが手に入るまでの繋ぎで試してみて下さい。
